# Biswajyoti Nath — Academic Research Portfolio

A high-performance, aesthetically refined academic portfolio designed for an undergraduate researcher specializing in **Symbolic Regression, Evolutionary Computation, and Interpretable Machine Learning**. 

Built with a "Coffee & Cream" minimalist aesthetic, utilizing hardware-accelerated fluid animations and progressive glassmorphism.

## Architecture & Stack
- **Frontend Framework:** React 18, Vite
- **Styling:** Tailwind CSS (Custom "Coffee & Cream" color tokens)
- **Motion & Physics:** Framer Motion (Parallax, scroll-scrubbed text, magnetic interactions)
- **Typography:** Lora (Serif) & Inter (Sans-serif)
- **Deployment:** GitHub Pages (SPA static routing)

## Design Philosophy
This portfolio strips away common visual clutter in favor of a clean, academic aesthetic paired with highly deliberate, fluid interactions:
1. **Academic Focus:** Projects and open-source contributions are structured using a strict "Problem / Approach / Impact" framework for immediate readability.
2. **Fluid Motion:** Features Framer Motion-powered parallax depth, dynamic scroll-linked timelines, magnetic buttons, and progressive glassmorphism blurs that bring the page to life without overwhelming the content.
3. **Mobile-First Optimization:** Fluid animations and responsive layouts adapt seamlessly to mobile, with heavy physics-based interactions gracefully disabling on touchscreens to preserve UX.

## Local Development Initialization

```bash
# 1. Clone the repository
git clone https://github.com/biswajyoti-nath/biswajyoti-nath.github.io.git
cd biswajyoti-nath.github.io

# 2. Install dependencies
npm install

# 3. Initialize the development server
npm run dev
```

## Build & Deployment
The system is configured for deployment via GitHub Pages.

```bash
# Generate the minified production build
npm run build

# Preview the minified output locally
npm run preview
```
